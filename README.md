# Edu-Cove 教学材料网站

一个提供精品课程和精品作品的在线教育平台。

## 项目简介

Edu-Cove 致力于为用户提供高质量的教学资源，包括：
- 📚 精品课程
- 🎨 精品作品展示
- 💡 学习资源和教程

## 技术栈

- **前端框架**: React 19
- **构建工具**: Vite 8
- **代码检查**: ESLint

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173/

### 生产构建
```bash
npm run build
```

### 其他命令
```bash
# 预览生产构建
npm run preview

# 代码检查
npm run lint

# 清理构建产物
npm run clean
```

## 项目结构
